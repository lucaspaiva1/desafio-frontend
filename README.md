## Running APP

#### Setup environment

```bash
cp .env.example .env
```

#### Install app dependencies.

```bash
yarn
```

#### Run

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

```bash
yarn start
```

### `yarn start`
